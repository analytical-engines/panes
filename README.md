# Panes

![Default window image](docs/images/default_window.jpg)

An image viewer for macOS.

## Features

- ZIP / CBZ / RAR / 7z archive and image folder support
- Spread (two-page) and single-page view modes
- Multi-window with session restore
- Image catalog and content search
- Password-protected archive support

## Installation

Download the latest DMG from [Releases](https://github.com/analytical-engines/panes/releases).

> **Note:** This app is unsigned. On first launch, right-click and select "Open" to bypass Gatekeeper.

## Requirements

- macOS 15.0 (Sequoia) or later
